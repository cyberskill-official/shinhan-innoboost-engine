# Data Card — SVFC Consumer Finance

## Provenance

| Field | Value |
|---|---|
| **Source** | Synthetic (generated by CyberSkill data pipeline) |
| **Business Unit** | SVFC (Shinhan Vietnam Finance Company) |
| **Generation Method** | Seeded pseudorandom with Faker-VN |
| **Seed** | 42 (default) |
| **Reproducible** | Yes — same seed produces identical output |
| **Contains Real PII** | No — all identifiers use `SYN-` prefix |
| **Total Rows** | ~20M+ |
| **Tables** | 8 |

## Tables

| Table | Rows | Description |
|---|---|---|
| `branches` | 32 | SVFC branches across 32 Vietnamese provinces |
| `products` | 12 | Consumer finance products (personal, vehicle, housing, credit card, SME, etc.) |
| `customers` | ~50,000 | Consumer finance customers with KYC status |
| `loans` | ~120,000 | Loans with NPL distribution ~2.5%, includes dual-currency edge cases |
| `payments` | ~2,000,000 | Payment records with principal/interest split |
| `risk_scores` | ~18,250,000 | Daily risk score snapshots per customer (365 days) |
| `collections_actions` | ~15,000 | Collections activity on overdue loans |
| `marketing_campaigns` | 50 | Marketing campaign performance |

## Sensitivity Classification

- **Public**: branch, product, campaign aggregates
- **Internal**: collections notes
- **Restricted**: customer name, phone, email, DOB
- **Regulated**: CCCD (national ID)

## Edge Cases Seeded

- ~2.5% NPL (overdue loans with DPD 30–180)
- ~0.3% written-off loans (DPD 180–365)
- ~0.2% restructured loans
- ~2% KYC-flagged customers
- ~1% dual-currency (USD) loans
- Seasonal patterns in origination dates

## Known Limitations

- Payment records are simplified (no late-fee logic)
- Risk scores use random walk, not actual credit model
- Collections outcomes are uniformly distributed
